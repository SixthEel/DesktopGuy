How to Use

1. Open Manager

Launch the GIF Overlay Manager application

2. Load GIFs

Click "Load GIF" to select your GIF files

3. Adjust Settings

Select a GIF from the list and adjust scale if needed

4. Display GIF

Click "Show Selected" to display the GIF on your screen

5. Control Mode

Use the checkbox to toggle click-through mode

6. Reposition

Drag GIFs with left mouse button to move them around

7. Quick Close

Press Ctrl+Q to close all GIFs at once
